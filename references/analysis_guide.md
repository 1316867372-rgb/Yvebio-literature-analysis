# 8模块拆解撰写指南

## 质量要求

每个模块必须达到充分深度，不可只用1-2句话概括。整体文档纯文本目标5000-8000字。

## 一、关键要素

- **疾病**：从Title/Abstract提取疾病全称并展开描述（1-2段，简述临床意义）
- **表型**：细胞层面的变化，不带方向词。平行嵌套用"+"连接，因果嵌套用"→"连接。需描述每种表型的生物学意义
- **主变量**：注明类型（蛋白/RNA/DNA），描述其已知功能和在相关疾病中的报道，点明本研究的创新性。如果有交互变量和效应变量同样详述
- 附带：机制模式（套路类型+直接机制+间接机制，每项展开描述）+ 研究模型（细胞/动物/临床，含具体细胞系名称、动物模型类型、临床样本量和来源）

## 二、科学假设

完整嵌套式假设句：在【疾病】中，主变量【X】（类型）通过与交互变量【Y】（类型）结合，调控效应变量【Z】，从而影响【表型】。

如无交互变量，则省略该部分。

## 三、逻辑关系

逐条列出因果逻辑链，每条附对应图号。每条至少包含：作用方向（上调/下调/抑制/促进）+ 实验方法 + 对应图号。

## 四、知识背景

按5个维度拆分，每项至少3-5个自然段：
1. **疾病机制**：临床问题、病理机制、流行病学数据、当前治疗困境、引用文献支撑
2. **表型概念**：定义、关键标志物、在疾病中的角色、相关分子调控网络
3. **主变量功能**：蛋白结构特征、已知生物学功能、在相关疾病中的报道（分类描述）、本研究的创新/空白点
4. **效应通路**：核心信号轴的分子机制（详细展开，如通路ON/OFF状态的分子事件）、上下游调控关系、在疾病中的作用
5. **交互蛋白**：如有交互变量，描述其结构功能、与主变量的已知关联、在疾病中的功能报道。如无交互变量，可合并入效应通路或省略

## 五、研究流程

按Fig顺序，每步详述：实验目的+所用方法+分组信息+推进逻辑。不可一句话带过。每个Fig至少2-3句话。

## 六、数据解读

**最关键模块，必须逐子图详读。**每个Fig独立一节，格式：

```
Fig X. 中文标题
【实验层次 | 论证类型 | 论证规范标注】
实验目的：详细描述为什么要做这个实验。
实验分组：列出所有组别。
A. 方法+具体结果+统计量(p值)。（论证规范标注）
B. 方法+具体结果+统计量(p值)。（论证规范标注）
...
结论：总结该Fig的核心发现。
论证评价：评估论证是否充分（完成/缺什么）。
```

每Fig数据解读至少200-400字。不可用一句话概括。子图需逐条覆盖，不可合并跳过。

## 七、论证规范

分三个子部分：

### （一）理论上应完成的论证
根据因果逻辑逐项推导完整的论证要求，每项用独立段落。包括：
1. 单变量论证（表达差异+一正一反+细胞动物）
2. 调控关系论证（两两调控+三三回复，拆分为具体条目）
3. 交互关系论证（如有交互变量，含双向互作+位点解析+必要性论证）

### （二）文献实际完成度
逐项对照理论规范，标注：完成(+)/部分完成(+/-)/完全缺失(--)。每项附具体说明和完成度百分比评估。不可只写"完成度约X%"，必须解释为什么。

### （三）论证规范总评
一段概括性评价，总结主要优势和关键缺口。

## 八、套路归纳

**优缺点必须各至少5条**，每条独立段落，附具体分析而非简单罗列关键词。

优点从以下角度选评（不需要全用）：
- 创新性（主变量、交互关系、表型选择）
- 故事完整性（上游→主变量→下游→表型的链条完整性）
- 论证完整度（正反验证、双向交互、位点解析、回复实验）
- 研究深度（是否做了位点/修饰层面分析）
- 论证广度（是否多细胞系、多动物模型、临床组织多维验证）
- 机制多元化（是否多套路嵌套）
- 讨论质量（是否诚实面对局限性）

缺点同样选评：
- 单变量论证缺失（缺正/缺反/缺动物/缺临床）
- 交互论证不足（缺双向/缺位点/缺必要性）
- 机制浅（停留在相关性未进入因果性，缺回复实验）
- 模型单一（仅一种细胞/无动物/临床样本量小）
- 统计效力问题
- 逻辑缺口（上下游因果关系未闭合）
